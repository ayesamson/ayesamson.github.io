PowerShell Tools and Scripts

I've taken to PowerShell and haven't looked back. I started with vbs, cmd, batch and WSH. In many cases I had written lots of scripts with many that blended the various and available technologies at the time. Some handled the job better than others but in some cases you need a little bit of this and that.

PowerShell has changed the game in my opinion. And that's probably safe to say for a lot of others. Everything I wanted to do that required the different technologies can now be accomplished with just PowerShell. That's just bloody fantastic! I'm working on my github repository and that should be available soon.
